## 题目简述

求 `1+2+...+n` ，要求不能使用乘除法、`for`、`while`、`if`、`else`、`switch`、`case`等关键字及条件判断语句`（A?B:C）`

class Solution {
    public int sumNums(int n) {
        boolean flag = n > 0 && (n += sumNums(n - 1)) > 0;
        return n;
    }
}
